# 项目介绍

> **最后更新**: 2026-04-21

---

## 项目概述

基于 LangGraph + MiniMax + AKShare 的 A 股分析系统，支持 6 个专业 Agent 并行分析。

**核心约束**: P0 风险控制 - 完整免责系统，保留专业投资建议术语。

---

## 版本说明

| 版本 | 状态 | 支持功能 |
|------|------|----------|
| **V1** | 当前版本 | 单股分析、投资建议（买入区间/止盈/止损）、多渠道推送 |
| **V2** | 开发中 | 批量分析、定时调度 |

---

## 核心特性

- **6 个专业 Agent**：量化分析师、图表分析师、情报官、风险评估师、基本面分析师、舆情监控师
- **并行执行**：LangGraph Send API 实现 6 个 Agent 真并发
- **双重权重模式**：固定权重（Fixed）或 Supervisor 动态分配（Auto）
- **P0 合规**：保留专业投资建议 + 完整免责文本注入
- **多渠道推送**：飞书、Server酱（微信）、钉钉、控制台
- **投资建议**：报告包含买入区间、止盈区间、止损位（基于分析结论）

## Agent 架构

| Agent | 路由键 | 权重 | 核心职责 |
|-------|--------|------|----------|
| 量化分析师 | `quantitative` | 25% | 技术指标、资金流向、趋势判断 |
| 图表分析师 | `chart` | 15% | K线形态、支撑阻力位 |
| 情报官 | `intelligence` | 10% | 宏观政策、行业动态 |
| 风险评估师 | `risk` | 20% | 风险量化、**止损位计算** |
| 基本面分析师 | `fundamental` | 20% | 财报解读、估值分析 |
| 舆情监控师 | `sentiment` | 10% | 情绪分析、分析师评级 |

## 投资建议说明

系统报告包含基于数据分析的投资建议：

| 内容 | 说明 |
|------|------|
| 操作建议 | 建议买入 / 建议观望 / 暂不推荐 |
| 买入区间 | 当前价格附近的合理买入区间 |
| 止盈区间 | 基于压力位的目标价位 |
| 止损位 | 基于波动率止损和支撑位 |

**生成依据**：
- 综合评分 >= 70：建议买入
- 综合评分 50-69：建议观望
- 综合评分 < 50：暂不推荐
