## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

创建 `.env` 文件：

```bash
MINIMAX_API_KEY=your_api_key
WEIGHT_MODE=fixed  # 或 auto
```

### 3. 运行分析

```bash
# 单股分析（V1 唯一支持的模式）
python -m src.main --stock 000001.SZ --name 平安银行 --notify console
```

## 环境变量配置

### 必须配置

| 变量 | 说明 | 示例 |
|------|------|------|
| `MINIMAX_API_KEY` | MiniMax API 密钥 | `sk-xxx` |
| `MINIMAX_BASE_URL` | API 地址（默认无需修改） | `https://api.minimaxi.com/anthropic` |
| `MINIMAX_MODEL` | 模型名称（默认无需修改） | `MiniMax-M2.7` |

### 可选配置

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `WEIGHT_MODE` | 权重模式：`fixed`=固定权重，`auto`=自动权重 | `fixed` |
| `SCHEDULER_ENABLED` | 调度开关 | `true` |
| `SCHEDULER_TIME` | 调度时间 | `10:00` |
| `BATCH_TOP_N_ENABLED` | 批量分析是否仅输出前N名 | `true` |
| `BATCH_TOP_N_COUNT` | 批量分析前N名数量 | `5` |

### 通知渠道配置

| 渠道 | 配置项 | 说明 |
|------|--------|------|
| 控制台 | 无需配置 | 直接输出到终端 |
| Server酱 | `SERVERCHAN_KEY` | 微信推送 |
| 飞书 | `FEISHU_WEBHOOK` | 飞书群机器人 |
| 钉钉 | `DINGTALK_WEBHOOK` | 钉钉群机器人 |

详细配置见下方[通知渠道配置](#通知渠道配置)章节。

### Agent 权重配置（固定权重模式）

```bash
WEIGHT_QUANTITATIVE=0.25   # 量化分析师
WEIGHT_CHART=0.15           # 图表分析师
WEIGHT_INTELLIGENCE=0.10   # 情报官
WEIGHT_RISK=0.20           # 风险评估师
WEIGHT_FUNDAMENTAL=0.20    # 基本面分析师
WEIGHT_SENTIMENT=0.10      # 舆情监控师
```

## 常用命令

### 单股分析（V1 唯一支持的模式）

```bash
# 基本用法
python -m src.main --stock 000001.SZ --name 平安银行 --notify console

# 指定权重模式
python -m src.main --stock 000001.SZ --name 平安银行 --weight-mode auto

# 指定通知渠道
python -m src.main --stock 000001.SZ --name 平安银行 --notify serverchan
```

### 权重模式

```bash
# 固定权重（默认）- 使用配置中的固定比例
python -m src.main --stock 000001.SZ --weight-mode fixed

# 自动权重 - Supervisor 根据数据质量动态分配
python -m src.main --stock 000001.SZ --weight-mode auto
```

### 批量分析（V2 开放，V1 暂不开放）

```bash
# V1 提示：批量分析功能暂不开放
python -m src.main --batch --notify serverchan
```

### 定时调度（V2 开放，V1 暂不开放）

```bash
# V1 提示：定时调度功能暂不开放
python -m src.main --schedule --notify serverchan
```

## 通知渠道配置

### Server酱（微信推送）

1. 访问 [Server酱官网](https://sct.ftqq.com/)
2. 注册并登录，获取 SendKey
3. 配置环境变量：

```bash
SERVERCHAN_KEY=your_sendkey
```

### 飞书配置

1. 在飞书群中添加"自定义机器人"
2. 复制 Webhook URL
3. 配置环境变量：

```bash
FEISHU_WEBHOOK=https://open.feishu.cn/open-apis/bot/v2/hook/xxx
```

### 钉钉配置

1. 在钉钉群中添加"自定义机器人"
2. 复制 Webhook URL
3. 配置环境变量：

```bash
DINGTALK_WEBHOOK=https://oapi.dingtalk.com/robot/send?access_token=xxx
```

## 股票池管理

股票池用于 V2 批量分析功能，当前 V1 版本可提前配置。

```bash
# 查看股票池
python -m src.main stock list

# 添加股票
python -m src.main stock add --code 000001.SZ --name 平安银行 --tags 银行

# 移除股票
python -m src.main stock remove --code 600036.SH

# 启用股票（参与 V2 批量分析）
python -m src.main stock enable --code 000001.SZ

# 禁用股票（跳过 V2 批量分析）
python -m src.main stock disable --code 600036.SH
```

股票池数据存储在 `config/stock_pool.json`。

## 关键文件说明

| 文件 | 职责 |
|------|------|
| `src/main.py` | CLI 入口，参数解析，单股分析流程控制 |
| `src/graph/builder.py` | LangGraph 工作流构建，条件边路由，Send API 并行 |
| `src/graph/state.py` | AgentState、AgentResult、StockQuery 数据结构 |
| `src/agents/supervisor.py` | 协调者，汇总评分，生成报告（含投资建议） |
| `src/agents/risk_analyst.py` | 风险量化，计算止损位数据 |
| `src/compliance/disclaimer_filter.py` | 过滤高风险煽动性词汇，保留专业术语 |
| `src/compliance/mandatory_disclaimer.py` | 注入完整法律免责 |
| `src/notification/notifier.py` | 多渠道通知分发 |
